## Usage

### Bootstrap

Clone the sources and just call the CLI to create a Python Virtualenv, e.g.:

```bash
~$ git clone https://github.com/jedie/thinkerforge2mqtt.git
~$ cd thinkerforge2mqtt
~/thinkerforge2mqtt$ ./cli.py --help
```
